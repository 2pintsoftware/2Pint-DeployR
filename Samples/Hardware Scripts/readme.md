# Hardware Scripts

This section will include scripts for the major OEM hardware vendors to assist with managing their systems.

The Community has created some step definitions to help manage software updates (Drivers / BIOS) for some of the major vendors.

<https://garytown.com/deployr-importing-custom-step-definitions>
